# Emotion-Aware Study Companion

## 📌 Project Description
Emotion-Aware Study Companion is a web-based system that analyzes a student's typing behavior to understand their emotional and focus state while studying. The system detects patterns like typing speed, pause time, and backspace usage to estimate whether the student is focused, tired, or stressed.

Based on the detected state, the system provides feedback to help students improve their study productivity.

---

## 🎯 Objective
The objective of this project is to support students in maintaining better focus during study sessions by identifying emotional patterns through typing behavior and providing helpful suggestions.

## ✨ Features

- Automatic typing behavior detection
- Emotion prediction (Focused / Tired / Stressed)
- Live focus level chart
- Social media distraction warning
- Modern gradient UI design
- Emoji-based emotion feedback

---

## 🧠 System Workflow

1. User starts typing in the study input box
2. System records typing speed and pause duration
3. Behavior data is analyzed
4. Emotion state is predicted
5. System displays emotion and study suggestions

---

## 📊 Emotion States

😊 Focused – Student is actively studying  
😴 Tired – Student may need a short break  
😣 Stressed – Student might be overwhelmed
